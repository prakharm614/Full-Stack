# React Counter App

A simple counter application built with React to demonstrate basic state management and event handling.

## Features

- Increment counter value
- Decrement counter value
- Reset counter to zero

## Technologies Used

- React
- Vite
- JavaScript
- HTML/CSS

## Getting Started

1. Clone this repository
2. Run `npm install`
3. Run `npm run dev`
4. Open browser at the specified localhost port
